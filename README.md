# MyLibs

## Projeto final de estudos

Este projeto tem por finalidade consolidar o conhecimento obtido através dos cursos de Angular realizado na Alura.</br>
O projeto consiste em uma aplicação de estante virtual, onde o usuário adiciona livros, status de leitura e se o livro é uma opção de compra futura.</br>

O intuito é fazer a aplicação de forma simplista, mas atendendo os princípios de melhores práticas, e que seja funcional. 

## Técnologias 

- Angular
- json-server
